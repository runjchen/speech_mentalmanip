# Human Annotations  

This directory contains resources and documentation related to human annotations for the **speech_mentalmanip** project.  

## Contents  
- `README.md`: Overview of the directory.  
- `audio_files/`: Contains all raw `.mp3` files used for annotation.  
- `templates/`: Blank `.csv` files provided for annotation purposes.  
- `annotations_audio/text_compiled.csv`: Compiled results of the annotations for audio or text modalities.
- `annotator_agreement.ipynb`: Script to analyze the annotations and calculate inter-annotator agreement.
