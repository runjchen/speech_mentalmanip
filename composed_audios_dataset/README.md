# Composed Audio Dataset

This directory contains composed audio datasets generated from text conversations.

## Files

- **`text_conversations_mental_manipulative_composed_audios_batch_01.zip`** (295 audio clips): Each audio clip corresponds to a text conversation that has been labeled as mentally manipulative in the MentalManip dataset.

- **`text_conversations_mental_manipulative_composed_audios_batch_02.zip`** (314 audio clips): These additional audio clips expand the dataset with more examples of mentally manipulative conversations converted to speech.

- **`text_conversations_non_mental_manipulative_composed_audios.zip`** (90 audio clips): These serve as negative examples, representing normal conversations without mental manipulation tactics.

## Ground Truth Labels

The ground truth labels for the text conversations in these composed audio datasets are taken from the [MentalManip dataset](https://github.com/audreycs/MentalManip/blob/main/mentalmanip_dataset/mentalmanip_con.csv).
